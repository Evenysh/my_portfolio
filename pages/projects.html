Todo-приложение
Описание проекта:

Простое и эффективное приложение для управления задачами. Позволяет добавлять, редактировать, удалять и отмечать задачи как выполненные. Данные сохраняются в локальном хранилище браузера.

Технологии:
JavaScript LocalStorage
Функциональность:
  • Добавление/удаление задач
  • Отметка выполненных задач
  • Фильтрация по статусу
  • Локальное сохранение данных
